# Math Blocks — FR/NL (Composition & Soustraction sans "emprunt")

Deux jeux interactifs (6–8 ans) basés sur **composer / décomposer** :
- **Jeu 1 : Construire le nombre** (dizaines / unités, transformations 10 ↔ 10×1)
- **Jeu 2 : Soustraire = transformer** (on transforme une dizaine en 10 unités si nécessaire)

## Langues
- Français (par défaut)
- Néerlandais (NL)

Le sélecteur FR/NL est disponible en haut de chaque page.

## Démarrage
Ouvrez `index.html` dans un navigateur (aucune dépendance).

## GitHub Pages
1. Créez un repo et uploadez les fichiers.
2. GitHub → Settings → Pages → Deploy from branch → `main` / root.
3. Ouvrez l’URL fournie.

## Stockage
- Progression et langue enregistrées via `localStorage`.
